# BorrowBox

# 🚀 1. Концепт проекта

## Идея

Платформа, где люди могут **арендовать вещи друг у друга** (в первую очередь технику), с приятным интерфейсом.

## Позиционирование

**“Платформа для аренды техники и полезных вещей”**

## Пример:

* нужна камера на 2 дня → нашёл → выбрал даты → отправил запрос
* есть PS5 → выложил → получаешь заявки

# 🧠 3. Логика проекта

---

## 3.1 Основные сущности

### Item (объявление)

```js
{
  id,
  title,
  category,
  pricePerDay,
  city,
  condition,
  description,
  image,
  owner,
  deposit
}
```

---

### Booking (заявка)

```js
{
  id,
  itemId,
  startDate,
  endDate,
  totalDays,
  totalPrice,
  status // pending / approved / declined
}
```

---

## 3.2 Основные функции

* поиск по названию
* фильтр по категории
* сортировка
* добавление объявления
* добавление в избранное
* создание заявки
* изменение статуса заявки

---

## 3.3 Логика дат

* нельзя выбрать endDate < startDate
* считаем дни:

  ```
  days = end - start
  ```

---

## 3.4 Логика цены

```
totalPrice = days * pricePerDay
```

---

## 3.5 Избранное

* сохраняем id в localStorage
* проверяем, чтобы не дублировалось

---

## 3.6 Поток пользователя

### Арендатор:

* ищет → открывает → выбирает даты → отправляет запрос

### Владелец:

* добавляет вещь → получает запрос → меняет статус
